# Premium Mobile Navigation

A sleek, responsive mobile navigation bar built with **React**, **Vite**, and **Tailwind CSS**. Featuring a premium dark mode theme with a smooth, curved active indicator and gold accents.

![Preview](/assets/preview.png)

## ✨ Features

- **Premium Dark Design**: Zinc-based dark theme with amber/gold accents.
- **Fluid Animation**: Smooth transitions and curved cutout effects for active menu items.
- **Responsive**: Designed specifically for mobile-first user interfaces.
- **Modern Stack**: Built with React 19 and Vite 8 for optimal performance.
- **IonIcons Integration**: Clean and recognizable iconography.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v20.19+ or v22.12+ recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rajjitlai/Mobile_Navigation.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Mobile_Navigation
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Create a production-ready build:
```bash
npm run build
```
